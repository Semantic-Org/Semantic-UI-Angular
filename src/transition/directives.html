Transition Directives
Create & Control Semantic UI Transition with Angular Directives

sm-transition
Executes a transition when an event occurs on the element.

Attributes
  • sm-transition @ a string of the transition to play.
  • sm-transition-events @ a string of the events that trigger the transition.
  • sm-transition-other @ a selector string of the elements to transition when the event occurs.
